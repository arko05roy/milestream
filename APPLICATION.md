## Project name

MileStream

## Ticker

MSTR

## One-sentence description

MileStream is a contractor-credential Stack Asset on HACD paired with GrowStreams milestone payments so freelancers get paid for small gigs and clients can trust non-fake profiles.

## Category

Utility

## Problem

Freelancers on Telegram, Discord, and informal platforms deliver work and never get paid. "Verified" profiles are free to fake. Existing escrow is too heavy for $50–$500 gigs.

## Solution

- HACD hybrid token (MSTR + Contractor ID): formation cost filters spam, 200 capped lots
- GrowStreams: milestone payment streams (25% start, 75% delivery)

## Why HACD (not a normal token)

Formation cost and PoW-mined HACD containers make contractor credentials costly to fake. Each lot's unique 6-letter HACD name is a permanent Contractor ID on-chain.

## Asset type

HYBRID — fungible MSTR + unique Contractor ID per HACD name

## Supply

1,000,000 MSTR across 200 lots × 5,000 per lot
